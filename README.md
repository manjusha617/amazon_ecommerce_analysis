# Amazon E-commerce Analysis

## Project Overview
This project analyzes Amazon e-commerce data to identify trends in ratings, prices, discounts, and customer reviews.

## Objectives
- Analyze product ratings
- Study discount patterns
- Identify top-performing categories
- Explore relationships between price and ratings

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Google Colab

## Dataset
Amazon E-commerce Dataset

##Dashboard
-powerbi dashboard to create visualize
-rating distribution
-sales trends
-category performance
